## Portfolio of me
I built this little vue.js-App is for introducing myself.

## Tech/framework used
<b>Built with</b>
- [Webpack](https://webpack.js.org/)
- [Vue](https://vuejs.org/)
- [Sass](https://sass-lang.com/)

## Installation
Install the dependencies: *npm install*
Run the project in dev-mode: *npm run dev*
Build the project: *npm run build*

Open my [Portfolio](http://localhost:6268/#/) with a modern Browser.
